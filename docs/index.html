Get Started

Option 1: Quick Install

Download the pre-built binary for macOS 13+:

curl -L https://github.com/boska/vox/releases/download/v1.1.0/vox-1.1.0-darwin-arm64 -o ~/vox && chmod +x ~/vox

Option 2: Configure Claude Code

Add to ~/.claude.json:

{ "mcpServers": { "vox": { "command": "/Users/$(whoami)/vox" } } }

Option 3: Build from Source

Clone and build with Swift:

git clone https://github.com/boska/vox cd vox swift build -c release
